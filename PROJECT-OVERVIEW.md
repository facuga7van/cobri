# Project Overview

Cobri is a modern SaaS platform for managing recurring subscriptions and customers. Built with Next.js and Firebase, it provides real-time KPIs, revenue tracking, and comprehensive subscription management with multilingual support (Spanish/English) and a beautiful dark/light mode interface.

## Technologies Used

### Core
- **Next.js 14** - React framework with App Router for routing and server-side rendering
- **React 18** - UI library
- **TypeScript 5** - Type-safe JavaScript

### UI & Styling
- **Tailwind CSS 4** - Utility-first CSS framework
- **Radix UI** - Accessible component primitives
- **Lucide React** - Icon library
- **Recharts** - Chart library for revenue visualization

### Backend & Database
- **Firebase** - Backend as a Service
  - Authentication (Google OAuth)
  - Firestore (NoSQL database)
  - Real-time data synchronization

### Features
- **next-intl** - Internationalization (Spanish/English)
- **next-themes** - Dark/Light mode with system preference
- **React Hook Form + Zod** - Form handling and validation
- **date-fns** - Date manipulation and formatting
- **Sonner** - Toast notifications
